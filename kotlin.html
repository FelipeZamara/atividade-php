KOTLIN

É uma linguagem de programação moderna desenvolvida pela JetBrains, e foi oficialmente endossada pela Google como uma linguagem suportada para o desenvolvimento Android

-Vantagens:

É compatível com Java, permitindo o uso de bibliotecas existentes além de facilitar a migração gradual de projetos Java para Kotlin.
É muito concisa e expressiva resultando em códigos mais legíveis com uma menor quantidade de código para realizar tarefas especificas em comparação com Java.
É uma linguagem oficial pro desenvolvimento Android segundo a Google e por conta disso tem um suporte oficial com forte apoio e recursos dedicados pro uso de Kotlin nesse ecossistema.

-Desvantagens:

Por ser mais novo e menos estabelecido que o Java tem situações em que suas bibliotecas e ferramentas ainda não foram tao desenvolvidas e maduras quanto as do Java.
Os binários gerados pelo Kotlin são maiores que os gerados pelo Java oque resulta em um maior aplicativo final.
Falta de fonte de conhecimento: por ser uma linguagem relativamente nova, não há tanto recursos de aprendizagem no mercado.
